# Security Policy

## Supported Versions

We are committed to maintaining the security of this project. Security updates will be applied to the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please take the following steps:

1. **Do not** open a public issue or disclose the vulnerability publicly.
2. Send an email to [security@lapair-project.org](mailto:security@lapair-project.org) with a detailed description of the vulnerability.
3. Include any relevant information that would help us reproduce and understand the issue.

We take all reports seriously and appreciate your efforts to improve the security of our project. We will respond as promptly as possible to address the issue.
