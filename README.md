# lapair

**lapair** is a Python module that parses source code from multiple programming languages into a sophisticated Intermediate Representation (IR). The module preserves as much information from the source codes as possible, providing detailed documentation on the IR interfaces.

## Features

- **Multilingual Support**: Parses source code written in C, C++, Objective-C, Swift, Java, Kotlin, Rust, JavaScript, TypeScript, and Python.
- **Sophisticated IR System**: Represents syntactic and semantic structures of the supported languages. Extensible, maintainable, and language-agnostic. Preserves all possible information from source codes, including comments, annotations, and metadata.
- **Custom Parsers**: Custom parsers for each supported language, written without external parsing dependencies. Accurately converts source code into the IR, handling all language-specific features with robust error handling.
- **Parallel Processing**: Utilizes parallel processing to improve performance. Parses multiple files or projects concurrently.
